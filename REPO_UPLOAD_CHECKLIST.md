# GitHub Upload Checklist

Before making the repository public or submitting the link:

- Confirm `.env` is **not** uploaded.
- Confirm API keys or tokens are **not** present in any script, notebook, or commit history.
- Confirm `README.md` matches the final repository structure.
- Confirm the GitHub repository name and URL exactly match the link in the report.
- Confirm the report links to **one** clean repository URL.
- Confirm generated result files are uploaded only if you want them versioned.
- Confirm your paper's repo URL is clickable and not broken across lines.
- Confirm teammate names are spelled consistently across the paper and repository.
